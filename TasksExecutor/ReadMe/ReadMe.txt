Использует библиотеку Selenium и GeckoDriver для Firefox.

Как работает:

1)  Открывает страницу с задачами и ищет по поиску задачи с заголовками: "Сменить комиссионные в брони", 
"Сменить статус в брони", "Сменить процесс работы с комиссионными".
            
2)  После каждого поиска получает список ссылок на типа задачами
            
3)  Далее открывает каждую задачу по ссылке, берет оттуда описание.
Описание должно быть в формате:

Регион: Санкт-Петербург
Бронь № 123456 (или несколько номеров)
Поменять: Комиссионные
Поменять на: 1,5 (или статус брони)

4)  Получает из описания 
–  регион и сравнивает с Regions; 
–  номера броней (может быть указана одна бронь или несколько через запятые, через пробелы,
через пробелы с запятыми);
Проверяет, чтобы номер брони был шестизначным;
–  если меняется статус, то сравнивается с Statuses
–  если меняются комиссионные (могут быть записаны в виде процента через точку или запятую),
то проверяет чтобы они были не больше 10%;

5)  Выполняет запрос. Проверяет, есть ли в базе такие брони

6)  Ставит теги, время и закрывает задачу 

7)  Если какие-то брони вывалились с ошибкой, то пишет ссылки на задачи в Log\Bookings.txt в папке с программой
